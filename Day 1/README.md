📌 Topics Covered

1. 🖨️ Hello World

- Learned how to print output in Python using "print()"
- Understood basic program structure

print("Hello, World!")

---

2. ⌨️ Taking Input

- Used "input()" to take user input
- Learned type conversion using "int()" and "float()"

name = input("Enter your name: ")
age = int(input("Enter your age: "))

print("Hello", name)
print("Your age is", age)

---

3. 🔀 Conditionals (if-else)

- Implemented decision-making using "if", "elif", and "else"
- Used comparison operators

num = int(input("Enter a number: "))

if num % 2 == 0:
    print("Even number")
else:
    print("Odd number")

🧠 Key Learnings

- Python syntax is simple and readable
- Input is always taken as a string and needs conversion
- Conditionals allow programs to make decisions
